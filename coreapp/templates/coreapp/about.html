About the Application

As mentioned, this clustering app is a piece of the algorithm made in the project 'Computer Program for the Identification of Celestial Objects through the Analysis of Light Curves', here I would like to present a bit of the project.

Resume

Variable stars are one of the main sources of information for astronomy, the study of the variation of brightness given in light curves, among other data, can provide information about their system, like if they have exoplanets. In this work, we analyzed 36 systems with exoplanets identified from observations made by the NASA Kepler space telescope. In this analysis we look for patterns in the brightness variation that are repeated in these systems, which may be related to the existence of exoplanets. The patterns sought are morphological patterns present in the light curves of each system, the data are analyzed in time series forms, using techniques to find motifs, enabling temporal analysis of the phenomenon.

Conclusion

Using time series techniques, we implemented a computer program with which we extract patterns from light curves. We obtained good results in separating these patterns, considering the measured quality of the grouping performed. Based on this grouping, strategies that use machine learning in combination with algorithms that count how many times a subsequence occurs in a period of time can be used to classify light curves, among others.

These concepts also apply to other situations with morphological comparisons and/or time series, so the computational implementation performed can be used to search for patterns in other databases.
